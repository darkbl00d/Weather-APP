Простое веб-приложение для получения текущей погоды по названию города или с использованием геолокации. Разработано с использованием следующих технологий:

- React: для создания пользовательского интерфейса.
- WeatherAPI: для получения данных о погоде.
- CSS: для стилизации приложения.
- JavaScript: для реализации логики приложения.

Пользователи могут видеть текущую температуру, влажность и состояние погоды в удобном интерфейсе. Приложение адаптивно и подходит для использования на различных устройствах.
Вы можете добавить или изменить технологии в зависимости от тех, которые вы использовали в своем проекте.
